# Local Models

Place your `.glb` or `.gltf` 3D model files in this directory. 
By placing them here, they will be tracked in your Git repository and served directly by the frontend, so you can reference them by their path (e.g., `/models/my-model.glb`).

Update the `localModels` array in `src/pages/AdminDashboard.tsx` to include any new models you add to this folder!
